## 🔧 Technical Implementation

### WordPress Publishing Strategy:
1. **Check if page exists** by slug
2. If new: **Create minimal draft** (avoids ModSecurity blocking large POSTs)
3. **Update with full content** via PUT request
4. Set status to **publish**
5. **Add to navigation menu** (Primary Menu #4, under Communities #1720)

### ModSecurity Bypass:
- Two-step approach: draft → update
- User-Agent: `WordPress/6.0` header (required)
- Content wrapped in Divi shortcodes: `[et_pb_section][et_pb_row][et_pb_column type="4_4"][et_pb_code]...`
